Comprehensive Git Guide

Install and Setup Git

brew install git
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"
mkdir examplegit
cd examplegit
git init
touch index.html
echo "<!DOCTYPE html><html><head><title>Example</title></head><body><h1>Hello, world!</h1></body></html>" > index.html
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/examplegit.git
git push -u origin master

Update and Push Changes

git pull origin master
nano index.html
git add index.html
git commit -m "Updated index.html with new content"
git push origin master
